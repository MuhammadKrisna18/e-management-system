# Sistem Pemesanan Event (Week 8)

## Aturan Bisnis

### Event
- Tanggal selesai tidak boleh lebih awal dari tanggal mulai
- Kapasitas harus lebih dari 0
- Status awal adalah Draft

### Kategori Tiket
- Harga tidak boleh negatif
- Kuota harus lebih dari 0
- Total kuota tidak boleh melebihi kapasitas event

### Publikasi Event
- Event harus memiliki minimal satu kategori tiket
